# RatGame Leaderboard Setup

1. Create a Supabase project.
2. Open the SQL editor and run all SQL from `docs/leaderboard/supabase.sql`.
3. Add environment variables for the web client:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
4. Build/deploy as normal (GitHub Pages static hosting is supported).
